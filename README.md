<h1> Consoles-Project</h1>
<p>Consoles-Project é o meu primeiro projeto que eu criei do zero.</p>
<p>Esse projeto consiste em mostrar os consoles disponíveis e adicionar 10% de desconto no preço dos consoles.</p>
